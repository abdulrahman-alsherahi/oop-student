# 🎓 C++ Student Marks System

A simple C++ program that demonstrates basic **Object-Oriented Programming (OOP)** concepts.

---

## 📌 Features

✔ Create Student objects
✔ Store student names and marks
✔ Calculate the total marks
✔ Find the student with the highest mark

---

## 🧠 Concepts Used

* Classes
* Constructor
* Encapsulation
* Friend Functions

---

## 👨‍🎓 Example Students

| Name  | Mark |
| ----- | ---- |
| Ahmed | 75   |
| Ali   | 90   |
| Abdul | 95   |

---

## 💻 Example Output

```
Total marks = 260
Abdul is the first
```
